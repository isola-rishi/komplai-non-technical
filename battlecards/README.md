# Komplai Battlecards

Sales battlecards for competitive positioning against direct competitors.

## Files

1. **battlecard-template.html** - Template for creating new battlecards
2. **battlecard-design-spec.md** - Design specification and guidelines
3. **battlecard-komplai-vs-numeric-2026-02-03.html** - Example: Komplai vs Numeric

## Quick Start

Open any `.html` file in your browser to view the battlecard.

To generate PDF: Open in browser → Cmd/Ctrl + P → Save as PDF

## Design System

All battlecards follow **Komplai Design System v2.0**:
- Dark green (`rgb(2, 31, 14)`) hero sections
- Playfair Display for headlines
- Inter for body text
- Bright green (`rgb(69, 209, 99)`) for success indicators
